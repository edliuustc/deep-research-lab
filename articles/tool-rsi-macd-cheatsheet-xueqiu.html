RSI / MACD / 均线 一图读懂

深研Lab · 工具型内容 · 建议收藏

这是一篇工具型内容，不教你炒股，帮你读懂三个最常用的技术指标。下次看K线图时，打开这张卡对照着看。

━━━ ① RSI（相对强弱指标）━━━

一句话理解：RSI就是一根"温度计"，测量股价最近涨得有多猛或跌得有多狠，数值0-100。

核心数值：

🔴 RSI > 70 → 超买区（过热），短期涨太多，随时可能回调

🟡 RSI 30~70 → 中性区，多空均衡，正常运行

🟢 RSI < 30 → 超卖区（过冷），跌过头了，可能迎来反弹

实战用法：

❶ RSI跌破30后重新拐头向上，比单纯低于30更有参考价值

❷ RSI突破70后还能继续涨是常见的，别急着卖，等回落跌破70再观察

❸ RSI背离是高价值信号：股价新高但RSI没新高=顶背离（危险）；股价新低但RSI没新低=底背离（机会）

⚠️ 常见误区："RSI超过70就卖，低于30就买"。强趋势中RSI可以长时间停留在超买/超卖区，它是警示灯，不是交易按钮。

━━━ ② MACD（平滑异同移动平均线）━━━

一句话理解：两条"快慢线"的追逐游戏——快线追上慢线（金叉）=上涨动能增强，快线被甩开（死叉）=下跌动能增强。

核心信号：

🟢 金叉（DIF上穿DEA）→ 看涨信号，零轴之上的金叉更强

🔴 死叉（DIF下穿DEA）→ 看跌信号，零轴之下的死叉更危险

🟡 红绿柱 → 动能强度计。红柱越长=涨势越猛，绿柱越长=跌势越重。柱子缩短=趋势减弱

实战用法：

❶ 零轴之上金叉是最可靠的买入信号（主趋势向上+动能加速）

❷ 红柱由长变短时要警惕，即使没死叉，上涨动能也在衰减

❸ MACD底背离是抄底参考，但建议等金叉确认后再行动

⚠️ 常见误区：一看到金叉就买、死叉就卖。MACD是滞后指标，震荡市中金叉死叉频繁交替，反复打脸。一定要结合趋势方向。

━━━ ③ 均线 MA（移动平均线）━━━

一句话理解：过去N天收盘价的平均值连成的线，相当于市场的"平均成本"。股价在均线上方=大部分人在赚钱。

常用均线：

MA5/MA10 → 短期趋势（1-2周方向）

🟡 MA20 → 多空分界线（最重要！站上偏多，跌破偏空）

MA60 → 中期趋势（季度线）

🔴 MA120/MA250 → 牛熊分界线（半年线/年线，最后一道防线）

实战用法：

❶ 均线多头排列（MA5 > MA10 > MA20 > MA60）= 强势上涨趋势，顺势做多

❷ 股价回踩MA20不破，往往是上涨趋势中的好买点

❸ 均线粘合后发散是大行情前兆——向上发散往往是主升浪起点

⚠️ 常见误区："跌破均线就是利空"。短期均线被反复穿越很正常，关键看哪条均线、是否放量、以及收盘价而非盘中瞬间。

━━━ 📋 三指标联合判断速查表 ━━━

单一指标容易误判，三个一起看才靠谱

🔴 RSI>70 + MACD死叉 + 跌破MA20

→ 强烈卖出信号｜三重确认见顶，优先减仓

🟢 RSI<30 + MACD金叉 + 站上MA20

→ 强烈买入信号｜三重确认见底，可考虑建仓

📈 RSI 50~70 + MACD零轴上金叉 + 站上MA20

→ 偏多｜趋势向好，可持股或轻仓加

📉 RSI 30~50 + MACD零轴下死叉 + 跌破MA20

→ 偏空｜趋势走弱，观望或减仓

⚡ RSI>70 + MACD金叉 + 站上MA20

→ 强势但过热｜趋势很强但别追高，等回调

⚡ RSI<30 + MACD死叉 + 跌破MA20

→ 超跌但未企稳｜可能继续跌，等MACD金叉确认

⏸️ RSI 40~60 + MACD零轴缠绕 + 反复穿越MA20

→ 震荡观望｜方向不明，耐心等待突破

━━━ 🛠️ 学会这张卡后怎么用 ━━━

第一步：打开你关注的股票日K线图，调出RSI、MACD指标，确认MA均线都显示。

第二步：先看均线判断大方向。多头排列还是空头排列？站在MA20上方还是下方？

第三步：用RSI判断当前位置。是刚起步（50附近）、已经过热（70+）、还是过冷（30-）？

第四步：用MACD确认买卖时机。只在方向对的时候才跟信号走。

第五步：翻开速查表对照。三个信号方向一致=可信度高；互相矛盾=该观望。

⚠️ 技术指标是辅助判断工具，不是预测水晶球。实际操作中，一定要结合基本面分析、市场情绪、仓位管理综合决策。
